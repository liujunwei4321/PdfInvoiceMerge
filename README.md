开发环境：VS Studio 2017，使用itextsharp库
## PDF 发票合并工具

**功能**：
将多张PDF发票按每页一张、两张或四张合成一个单一的PDF文件，方便打印。
注意合并时会把PDF展平，表单等内容会扁平化。
如果选择一页合并一张的话，可以合并的东西不止发票，其它PDF文件一样合并。
如果选择一页合并多张的话，只会抽取每个PDF文件的第一页进行组装。
