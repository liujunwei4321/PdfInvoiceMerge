开发环境：VS Studio 2017，使用itextsharp库
## PDF 发票合并工具

**功能**：
将多张PDF发票按每页一张、两张或四张合成一个单一的PDF文件，方便打印。也可以将多个文件做成PDF portfolio


如果选择一页合并一张的话，可以合并的PDF不止发票，其它PDF文件一样合并，不过不支持指定页码合并。

如果选择一页合并多张的话，只会抽取每个PDF文件的第一页进行组装。
